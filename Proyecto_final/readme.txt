Descripción del Proyecto
Esta pagina web permitira a los usuarios registrarse para crear, gestionar y participar en eventos personales (como bodas, fiestas de cumpleaños, eventos familiares, etc) de manera sencilla. La plataforma ofrece una interfaz intuitiva donde los usuarios pueden organizar eventos, ver detalles, confirmar su asistencia y explorar la ubicación del evento mediante la integración de un api de un servicio de mapas. La aplicación está diseñada para facilitar la organización de eventos tanto personales como públicos.

Características Principales
Gestión de Eventos:
-Los usuarios pueden crear, editar y eliminar eventos.
-Cada evento incluye información relevante como el título, la descripción, la fecha, la hora y la ubicación.
-Se pueden establecer eventos privados (solo para invitados específicos) o públicos (abiertos a todos los usuarios registrados).

Participación en Eventos:
-Los usuarios pueden unirse a eventos y confirmar su asistencia.
-Los organizadores pueden ver la lista de participantes y recibir notificaciones cuando alguien se inscribe en su evento.

Integración con API de Mapas:
Los eventos incluyen una ubicación que se muestra mediante una API de mapas (como Google Maps o Mapbox), lo que permite a los usuarios visualizar la ubicación exacta y obtener direcciones.

Notificaciones y Recordatorios:
Integración con una API de notificaciones para enviar recordatorios a los participantes antes del evento.
